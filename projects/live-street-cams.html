Eyes on the City

Go to Project

By: Joshua Heagle

My first opportunity to demonstrate my skills appeared in the form on a unique challenge at my first developer job. We were releasing a website to promote and provide information on an city-wide art event.

We thought it would be really interesting to provide a view of a live city cameras from around the city. The live camera feed were being taken from the official city website.

The challenge this posed is that you have to actually be on the website to view the camera, otherwise there would be an accessed denied message instead of the camera feed.

Go to Application 
<head>
    <script>
        /* Use JavaScript to write the styles that add the camera view as a background image. I used the eleven best cameras views I found. */
        var camera = Math.floor(Math.random() * 12); // Select a random number for which camera to display each time.
        switch (camera) {
            case 0:
                document.write("<style>.live-cam { background: url('http://traffic.ottawa.ca/map/camera?id=41');} </style>");
                break;
                        ...
            case 10:
                document.write("<style>.live-cam { background: url('http://traffic.ottawa.ca/map/camera?id=46');} </style>");
                break;
            default:
                document.write("<style>.live-cam { background: url('http://traffic.ottawa.ca/map/camera?id=189');} </style>");
        }
    </script>
</head>
<body>
    <!-- Use an iframe to access the city site and cache access to the site. Set its height and width to 0 (we don't want it to display)  -->
    <iframe src="http://traffic.ottawa.ca/map/index" frameborder="0" height="0" width="0" style="width: 0; height: 0"></iframe>
    <div class="live-cam"></div>
</body>
