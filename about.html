By: Joshua Heagle

I first began programming in my last year of high school. The first programming I learned was with Turing for practice and then I moved on to creating applications with Visual Basic 6.0. The first program I wrote without guidance was a game of battleship with A.I.

When I decided to attend college it was because of my strong desire to invent things and know how things work that led me to the Computer Engineering Technology - Computer Science program.

One of the languages I work with the most is Java; it was used for most of my programming-related training. Along with Java, I am also fond of C programming because it feels a lot rougher and it has so much potential for lower-level programming such as embedded programming.

I am a very strong believer in Object-Oriented programming, and I will frequently solve problems by creating classes and helper classes so that I can use and re-use code whenever possible.
